Partner Keyboard prototype
Tap the editor → strip appears above the soft keyboard
Two tricks make this work on iOS Safari:
  • visualViewport tracks the soft keyboard's position. The strip's bottom is set to layoutHeight − (visualHeight + visualOffsetTop) so it pins flush above the keyboard.
  • pointerdown.preventDefault() on every strip key — without this, tapping a key blurs the textarea and dismisses the system keyboard.
Insertion:
  • textarea.setRangeText() preserves the native undo stack (swipe-down-to-undo still works).
  • Paired chars ((), [], quotes) insert both halves and land the cursor between.
